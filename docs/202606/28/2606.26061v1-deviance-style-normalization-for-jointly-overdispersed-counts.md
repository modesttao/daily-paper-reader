## 摘要
我们针对稀疏、联合过度离散的计数矩阵（即主导基于测序的生物化学测定的计数矩阵）引入了一种狄利克雷-多项（DM）偏差残差化。DM零模型将每个样本的计数向量视为具有固定总数的组成，并由一个标量浓度α₀控制过度离散，通过以观测到的样本总数为条件对独立的负二项特征计数进行条件化而精确得到——这使得DM成为标准逐特征过度离散计数模型的联合条件对应物。所得的变换保留了精确的稀疏性，每个非零条目的评估时间为常数，在单一计数上与多项分布残差一致，根据零模型所能容忍的过度离散程度收缩重复计数残差，并在α₀→∞时恢复为多项分布残差。相同的固定离散度比较原理通过广义DM和狄利克雷树多项分布扩展到有序和树形结构特征，从而得到一个单一的残差族，在共同的组成逻辑下涵盖了联合和逐特征计数零模型，并且计算量足够轻量，可以嵌入现有的稀疏处理流程。

## Abstract
We introduce a Dirichlet--multinomial (DM) deviance residualization for sparse, jointly overdispersed count matrices, the regime that dominates sequencing-based biochemical assays. The DM null treats each sample's count vector as a fixed-total composition with a single scalar concentration $α_0$ governing overdispersion, and arises exactly by conditioning independent negative-binomial feature counts on the observed sample total -- making the DM the joint conditional analogue of standard feature-wise overdispersed count models. The resulting transform preserves exact sparsity, evaluates in constant time per nonzero entry, agrees with multinomial residuals on singleton counts, shrinks repeated-count residuals according to the overdispersion the null tolerates, and recovers the multinomial residual as $α_0\to\infty$. The same fixed-dispersion comparison principle extends to ordered and tree-structured features via the generalized DM and the Dirichlet-tree multinomial, giving a single residual family that subsumes joint and feature-wise count nulls under a common compositional logic and is computationally lightweight enough to drop into existing sparse pipelines.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题背景**：基于测序的生物化学实验（如 scRNA-seq、scATAC-seq、CITE-seq、微生物组 16S、CRISPR 筛选等）生成稀疏的高维计数矩阵，具有**联合过度离散**（joint overdispersion）特性——样本内各特征计数不仅服从更大的方差，而且受限于固定的样本总读数，导致特征之间存在负相关。
- **现有方法不足**：
  - **特征级模型**（如负二项 NB）能处理每个基因的过度离散，但忽略样本内特征间的竞争关系（联合固定总数）。
  - **联合模型**（如多项分布 Multi）尊重固定总数，但无法捕捉重复计数导致的过度离散（假设独立采样）。
  - **密度式变换**（如 CLR、log1p）会破坏原始矩阵的稀疏性，引入伪计数。
- **核心目标**：提出一种**基于似然的、稀疏保留的、联合过度离散的规范化变换**，使得零模型能够同时处理组成结构和过度离散，并且计算高效、可扩展至有序或树形结构特征。

## 2. 提出的方法论

### 2.1 核心思想
- 以 **Dirichlet–Multinomial (DM) 分布**作为零模型：每个样本的计数向量服从固定总数 \(n_i\) 的 DM 分布，其浓度参数 \(\alpha = \alpha_0 \pi\)，其中 \(\pi\) 是全局经验组成，\(\alpha_0\) 是标量浓度控制过度离散程度。
- 通过**条件独立负二项特征计数于样本总数**得到 DM 分布（定理 2.2）：\(X_{ij} \sim \text{NB}(r_j, p_i)\) 独立，则给定 \(n_i\) 后，\((X_{i1},...,X_{iK})\) 服从 \(\text{DirMult}(n_i, r)\)。这建立了 DM 与常用特征级 NB 模型的条件对应关系。
- **残差变换**：对于每个非零条目 \(x = X_{ij}\)，计算
  \[
  c_{ij}^{\text{DM}} = \sum_{m=0}^{x-1} \log \frac{\alpha_0 x / n_i + m}{\alpha_0 \pi_j + m},
  \quad d_{ij}^{\text{DM}} = \operatorname{sgn}(x - n_i \pi_j) \sqrt{|c_{ij}^{\text{DM}}|}
  \]
  该变换**保持精确稀疏性**（零输入零输出），每个非零元素 O(1) 时间计算。
- **关键性质**：
  - 与多项残差在 singleton 上一致（命题 2.4）；
  - 对重复计数残差进行收缩（命题 2.5），收缩程度由 \(\alpha_0\) 控制；
  - 当 \(\alpha_0 \to \infty\) 时退化为多项残差（命题 2.6）。
- **扩展**：
  - **广义 DM (GDM)**：用于有序特征（如基因组坐标），每个阶段有独立的 beta-binomial 分裂；
  - **Dirichlet-树多项 (DTM)**：用于树结构特征（如微生物组系统发育树），每个内部节点有独立的 DM 因子。

### 2.2 技术细节
- **浓度估计**：用固定点迭代或牛顿法优化 \(\alpha_0\)，基于 digamma/trigamma 函数，时间复杂度 O(nnz + n)。
- **拟合方案**：默认固定 \(\pi\) 为全局组成，仅拟合一个标量 \(\alpha_0\)；也提供联合拟合 \(\alpha_0,\pi\) 的选项用于敏感性分析。
- **e-变量与 e-过程**：通过行拆分（binomial thinning）构造 split-LR e-变量，实现有限样本、任意有效时间的假设检验（推论 2.10）。

## 8. 不足与局限

- **实验覆盖的局限性**：
  - **大规模数据验证不足**：单细胞数据最大为 20,000 细胞（GTEx 子集），未在更大规模（如百万细胞）上测试；推荐系统仅用 MovieLens 小数据集；NLP 仅用 20-Newsgroups 等中型语料。
  - **缺乏与最新端到端模型的公平对比**：在 NLP 和推荐系统任务中，仅与简单基线（TF-IDF, log1p）比较，未与 BERT、GPT、矩阵分解 ALS 等高性能模型相比。
  - **深度基线未充分运行**：scVI 训练时部分数据集（肺图谱）发散，且其 FDP 控制较差，但论文仍将其纳入对比，可能低估其潜在优势（若调优得当）。
- **潜在偏差风险**：
  - **固定组成假设**：默认使用全局经验组成 π，若样本有强批次效应，该假设可能过强；论文建议敏感性分析中使用联合拟合，但未提供系统指导。
  - **结构化扩展依赖先验**：GDM 和 DTM 需要已知的顺序或树结构，若结构错误（如随机置换）则表现下降，论文指出此时平坦 DM 更稳健。
  - **e-变量构造依赖行拆分**：每行需至少 2 个计数才能使用 split-LR，否则退化为平凡因子；在极小样本场景（如行总和为 1）适用性受限。
- **应用限制**：
  - 对于非组成性计数（如总计数也含信号，例如 RNA 表达量峰度）不适用，因为该变换固定总数只会移除深度变异。
  - 无法捕获 log-ratio 上的任意协方差结构，在强依赖关系（如逻辑正态模型）下单个 α₀ 的近似可能不够精确（文中显示 FPR 可能上升至 0.17，虽仍远优于多项分布）。
- **计算资源描述不足**：未提供 GPU 或 CPU 具体型号、核心数、内存量等，无法复现精确运行时。
