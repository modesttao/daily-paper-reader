## 摘要
高通量微流控活细胞成像生成了丰富的单细胞数据。然而，用于定位感兴趣区域（RoIs，每个区域包含一个细胞群）并从记录图像中去除周围微流控结构的半自动化程序，其工作量随RoI数量增加而增加。这阻碍了实时图像分析，并将洞察获取时间延迟数小时至数天。我们提出了一种用于微流控培养芯片的面向设计与实时处理（DART）范式，该范式将CAD蓝图与物理芯片对齐，从而实现了所有RoI的不依赖通量的定位，以及跨不同RoI几何形状和芯片布局的全自动化图像处理。DART通过嵌入的基准标记和基于深度学习的标记检测建立了这种对齐。我们使用瑞士军刀芯片验证了DART，该芯片在1164个RoI位置上组合了八种结构不同的RoI设计。DART在五分钟内定位所有RoI，在40毫秒内从原始显微图像中去除微流控结构，并在每张图像不到1.1秒内执行包括细胞分割在内的全自动图像分析。这些能力共同使DART成为一种端到端的硬件-软件范式，具备实时分析能力，为闭环和结果驱动的智能显微镜铺平了道路。

## Abstract
High-throughput microfluidic live-cell imaging generates rich single-cell data. Yet semi-automated procedures for locating regions of interest (RoIs), each containing one cell population, and removing surrounding microfluidic structures from recorded images, scale with the number of RoIs. This prevents real-time image analysis and delays time-to-insight by hours to days. We introduce the Design-Aware and Real-Time capable (DART) paradigm for microfluidic cultivation chips, which aligns the CAD blueprint with the physical chip and thereby enables throughput-independent localization of all RoIs and fully automated image processing across diverse RoI geometries and chip layouts. DART establishes this alignment through embedded fiducial markers and deep-learning-based marker detection. We validate DART using the Swiss Army Knife chip, which combines eight structurally distinct RoI designs across 1164 RoI locations. DART localizes all RoIs in five minutes, removes microfluidic structures from raw microscopy images in 40 ms, and performs fully automated image analysis, including cell segmentation, in under 1.1 s per image. Together, these capabilities establish DART as an end-to-end hardware-software paradigm with real-time-capable analysis that paves the way toward closed-loop and outcome-driven smart microscopy.

## 1. 核心问题与整体含义（研究动机和背景）
- **现状**：高通量微流控活细胞成像可生成丰富的单细胞数据，但现有工作流依赖大量手动或半自动步骤。
- **两大瓶颈**：
  - **实验设置**：手动定位每个感兴趣区域（RoI），耗时随RoI数量增长（可长达数小时），限制了高密度芯片设计的可扩展性。
  - **图像分析**：需手动或半自动去除微流控结构、排除无关区域，同样随通量增加，且延迟数天才能获取定量洞察。
- **目标**：实现实时、全自动的图像分析，克服上述瓶颈，为闭环和结果驱动的智能显微镜铺路。

## 2. 方法论
### 核心思想
- 将CAD蓝图与物理微流控芯片对齐，使芯片几何结构在实验时可直接用于图像分析，实现“设计感知”（design-aware）处理。
- 通过直接在芯片设计上嵌入基准标记（十字+圆点）和唯一ID，建立蓝图与物理设备之间的明确映射。

### 关键技术细节
1. **标记检测**：使用YOLO（You Only Look Once）深度学习模型（最终选型：YOLOv26-s，1280px输入，检测任务），检测图像中的十字和圆点标记，达到亚微米精度（95百分位误差0.18 μm），推理时间~16.4 ms/图。
2. **粗对齐（粗校准）**：
   - 手动记录3个参考RoI的图像、ID和舞台坐标（约5分钟人工操作）。
   - DART检测标记后，用仿射变换将蓝图坐标映射至舞台坐标系，从而一次性定位所有RoI（不依赖RoI数量）。
3. **细对齐（精校准）**：
   - 针对每张图像进行5步处理：
     1. 检测标记（YOLO）
     2. 标记配对与旋转角度估计（基于蓝图期望间距过滤）
     3. 图像旋转至蓝图坐标系（使用Kornia或OpenCV）
     4. 根据RoI ID加载蓝图掩膜并应用，裁剪并排除微流控结构
     5. 在裁剪后的RoI内进行细胞分割（Cellpose-SAM）
   - 自动错误检测：标记间距不符则丢弃该图像，防止错误掩膜传播。
4. **误差来源**：舞台定位误差、制造公差、PDMS变形等。

## 3. 实验设计
- **芯片设计**：瑞士军刀（SAK）芯片，含1164个RoI，涵盖8种不同几何结构（如常规盒、大盒、开放盒、母机陷阱等），包含内部微流控特征（支柱、收集器等）。
- **标记检测数据集**：319张显微图像，标注了472个十字标记和470个圆点标记，分为训练（60%）、验证（15%）、测试（25%）。
- **模型基准**：比较YOLO v5、v8、v11、v26，模型大小（n/s），输入分辨率（640/1280），任务（检测/分割）。主要指标：标记中心误差（中位数和95百分位）、推理时间。
- **粗对齐验证**：计算所有RoI的预测中心与图像推导中心之间的L2距离（舞台坐标），报告中位数和90百分位误差。
- **细对齐验证**：在C. glutamicum活细胞实验中处理1739张图像，统计各步骤处理时间。
- **对比方法**：无外部对比基线，主要与自身基准测试以及理论手动画线比较。

## 6. 主要结论与发现
- **粗校准**：5分钟人工操作即可定位全部1164个RoI，中位定位误差10.46 μm（约FoV宽度的6.4%），90百分位18.39 μm（17.6% FoV）。
- **细校准**：图像处理（不含分割）平均39.2 ms/图（~25.5 fps），含分割平均1.067 s/图（~0.94 fps），快于实验采集频率（~2 s/图），实现实时处理。
- **活细胞验证**：成功提取细胞总面积的生长曲线，拟合逻辑增长模型获得生长速率，处理1739张图像用时31分钟，识别超过50万个单细胞，实时因子17.4。
- **DART通过恢复芯片的结构设计知识，实现跨几何形状、通量无关的自动分析，为闭环智能显微镜提供了关键基础。**

## 7. 优点
- **设计感知（Design-aware）**：直接利用CAD蓝图中的先验信息，无需为每种RoI设计单独训练或手工掩膜。
- **通量无关**：粗校准工作量固定（3个参考点），不随RoI数量增加，可扩展至高密度芯片。
- **实时能力**：处理时间低于图像采集间隔，支持在线分析。
- **错误检测机制**：自动识别标记检测失败，防止错误传播。
- **开源与模块化**：软件开源，易于集成和扩展。
- **硬件适用性广**：使用标准微流控制造工艺，标记设计简单。

## 8. 不足与局限
- **粗校准精度受限**：90百分位误差达18.39 μm（17.6% FoV），对精细结构（如母机陷阱、1 μm宽通道）可能导致掩膜偏移或部分遮挡。
- **分割仍是瓶颈**：Cellpose-SAM耗时约1秒/图，限制更高帧率应用；仅测试一种分割模型。
- **泛化未充分验证**：仅在一个芯片（SAK）和一种菌种（C. glutamicum）上验证，未能证明在其他芯片布局、细胞类型或成像模式（如荧光）上的可迁移性。
- **无外部基准对比**：未与DeLTA 2.0等已有半自动方法进行定量性能对比。
- **制造与变形误差**：PDMS软材料在长实验中可能变形，影响对齐精度；标记可能因制造缺陷不可见。
- **标记设计限制**：需在芯片上额外占用空间，对非常紧凑的RoI布局可能不适用；标记尺寸需大于1 μm以避免制造困难。
- **实验重复性**：未报告多次独立实验重复结果，单次实验可能引入偏差。
